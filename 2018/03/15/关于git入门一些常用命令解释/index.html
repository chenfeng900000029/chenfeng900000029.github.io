关于git入门一些常用命令解释

2018.03.15 陈枫  热度 ℃ 
假如我们还没有本地仓库的情况下我们需要创建本地仓库

cd desktop    # cd 进入桌面 #    

mkdir myBlog  # mkdir 是创建目录 关于git入门一些常用命令解释 #

cd myBlog     # 然后我们就在myBlog里边嚯嚯 #

git init      # 初始化myBlog目录 目录内会多一个隐藏文件 .git #

ls -la          # 查看当前目录内所有文件 -l（会展现文件的详细信息 包括最近的修改时间 一些权限 
                  文件大小等相关内容） -a（显示当前目录隐藏文件）#
